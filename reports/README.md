# Reports

Each generated run should create a folder under `reports/runs/<theme-slug>/`.

Useful run artifacts include:

- `run-metadata.md`
- `plan.md`
- `ollama-*-prompt.md`
- `ollama-*-raw.md`
- `codex-*-prompt.md`
- `codex-*-raw.md`
- `validation-output.txt`
- `release-summary.md`

These reports are meant to make the factory debuggable after a failed or partial generation run.
