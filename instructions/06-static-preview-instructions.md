# Static Preview Instructions

- Mirror the WordPress theme visually using only local assets.
- Keep the HTML accessible and the layout representative of the theme.
- The preview must work without PHP or WordPress.
- Link the preview from `docs/index.html`.
- Include `index.html` plus `homepage_preview.html`, `services_preview.html`, `about-us_preview.html`, `contact_preview.html`, `single_services_preview.html`, `blog_preview.html`, and `work_preview.html`.
- Match the WordPress templates closely: same header, footer, class names, section order, copy style, image assets, buttons, and card layouts.
- Header links must navigate between the static preview files.
- Implement Nolan-menu behavior in local preview JS.
