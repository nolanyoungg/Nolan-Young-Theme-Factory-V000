# Design Style Instructions

- Build a coherent visual system with strong hierarchy.
- Use local industry-specific images, local graphics, gradients, patterns, or SVGs for polish.
- Give the hero distinctive composition, not just large text: pair a clear headline and CTAs with relevant local imagery, proof markers, visual texture, and responsive framing that fits the industry.
- Give the footer real design attention: it should close the page with brand confidence, organized links, contact or conversion paths, and refined spacing instead of feeling like a default afterthought.
- Make buttons, cards, forms, and hover states feel complete.
- Keep mobile behaviour deliberate and legible.
- Avoid generic agency defaults, gray boxes, placeholder image areas, and thin homepages.
- Aim for an elevated visual system that fits the prompt and industry: premium spacing, clean type, strong image composition, and refined detail.
