# Nolan Young Theme 006 - Lumora Harbor Hotel

Classic WordPress theme for a refined waterfront boutique hotel.

## What is included

- Full classic theme template set
- Local SVG visual assets for the hotel experience
- Runtime CSS and JavaScript in `assets/css/theme.css` and `assets/js/theme.js`
- Build script that copies runtime assets to `bundle.css` and `bundle.js`
- Matching static preview in `docs/themes/006_nolan_young_theme_lumora_harbor_hotel/`

## Build

```bash
npm install
npm run build
```

## Notes

- The theme uses local assets only.
- The header follows the Nolan-menu contract.
- The preview mirrors the WordPress sections and link structure.
