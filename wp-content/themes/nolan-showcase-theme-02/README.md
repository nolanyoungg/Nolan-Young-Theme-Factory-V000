# Nolan Showcase Theme 02

Nolan Showcase Theme 02 is a premium classic WordPress theme for a boutique photography studio. It pairs an editorial layout with warm cinematic imagery, responsive sections, and a static preview that mirrors the WordPress output.

## Included Experiences
- Home
- About Us
- Services
- Work
- Blog
- Contact
- Policies

## Build

```bash
npm install
npm run build
```

The build writes:
- `assets/css/bundle.css`
- `assets/js/bundle.js`

## Notes
- Uses local assets only.
- Avoids remote fonts, scripts, and images.
- Includes a static preview in `docs/themes/nolan-showcase-theme-02/`.
